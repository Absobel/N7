6.2 Vérifier la version de Java : Au plus JDK 11

Pour savoir quelle version du JDK est utilisée par Eclipse, on peut faire Window > Preferences puis choisir Java / Installed JREs. Ce doit être une version 11 au plus. Attention, il ne faut que ce soit une version supérieure même si le mode compatibilité 11 est activé !
