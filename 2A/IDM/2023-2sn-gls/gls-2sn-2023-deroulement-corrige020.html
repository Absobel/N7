6.4 Comment tester plus rapidement ma transformation Acceleo : déployer un greffon est long.

Pour tester une transformation dans l’eclipse de développement on peut faire « run Configuations » (depuis le menu contextuel « run as » par exemple). Il faut alors double-cliquer sur Acceleo Application et renseigner les champs de l’assistant qui apparaît (voir cette capture d’écran30) : le projet (qui contient la transformation), la transformation elle-même (cliquer sur search en fin de ligne listera les transformations disponibles), le modèle sur lequel appliquer la transformation (un modèle, pas un métamodèle !) et le dossier dans lequel engendrer les fichiers produits par la transformation (src-gen par exemple).

On peut ensuite rapidement relancer la transformation en utilisant le bouton vert d’exécution (lecture) sur la barre de bouton. On peut choisir l’exécution à effectuer en utilisant la petite flèche à la droite de ce bouton vert.
